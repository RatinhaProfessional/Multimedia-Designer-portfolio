University Project

The informational website about Polestar Precept progress tracked by YouTube documentary series. Polestar is a Swedish automotive brand acquired by Volvo in 2015 and currently leading the Precept concept of more sustainable future of motorization. The website was one of projects conducted during Multimedia Design study program focused on responsivity, mobile version mock-up was supplied and the design of desktop version was individually created. After the prototyping phase the mock up was implemented in code with use of mobile first approach.

Provided mock-up contained colour scheme,
font type and ready to use content in form
of photos and text.

Implementation of the design required use of CSS grid system and flex display as well as optimisation for technical SEO and semantic structuring.
While working on the desktop version design I established two media queries (500px and 1050px), that will efficiently work on smartphones, tablets
and smoothly transition into the desktop version.
